Cube in Square

Cube in Square is a Minecraft roleplay server and a toolkit site built around the same idea: creating memorable stories without mods.

What you can find here

Minecraft tools

We publish browser-based tools for resource packs, custom potions, villager trades, and other workflows that matter on a mod-free server.

Wiki and guides

Every important tool is paired with articles and step-by-step documentation, so the site works as both a builder and a reference library.

Project updates

The site grows together with the server, which means new tools and articles appear when they become useful in our real development and play sessions.

About the project

Stories without mods

Our goal is to build interesting shared stories with players while keeping Minecraft vanilla at its core.

Browser-first workflow

The main tools on this site run in the browser. PNG textures and generated ZIP files are handled locally on the user's device instead of being uploaded to the site server during normal use.
