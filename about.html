О нас

Universal MC Pack — небольшой браузерный проект для тех, кто хочет собирать ресурс-паки Minecraft без установки программ и без тяжелой технической обвязки.

Принципы проекта

Легкий код

HTML, CSS и JavaScript остаются простыми файлами, которые можно открыть локально или выложить на GitHub Pages.

Практичные инструменты

Каждая страница должна помогать собрать, проверить или понять ресурс-пак, а не прятать работу за лишними зависимостями.

Расширяемость

Новые страницы и утилиты добавляются отдельными файлами, чтобы проект было легко поддерживать и развивать дальше.
