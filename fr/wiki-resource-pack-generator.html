Comment utiliser le générateur de packs de ressources Minecraft

Créez un pack de ressources Minecraft directement dans le navigateur. Importez des textures PNG, choisissez ce qu’elles remplacent, prévisualisez le résultat et téléchargez un ZIP prêt.

1. Nommez le pack

Saisissez un nom de pack en haut. Il sera utilisé dans pack.mcmeta et comme nom du ZIP téléchargé.

2. Choisissez la version du serveur

Le sélecteur de version contrôle si les commandes give utilisent l’ancienne syntaxe NBT ou la nouvelle syntaxe Data Components 1.20.5+.

3. Ajoutez des objets

Open the Objets tab, enter a vanilla item ID, upload a PNG texture, and optionally enter a CustomModelData number for a custom variant.

4. Ajoutez des blocs

Open the Blocs tab to replace a block texture or build a cube from separate top, bottom, front, back, and side textures.

5. Ajoutez une armure

L’onglet Armure permet de remplacer les couches d’armure et les icônes d’inventaire des matériaux courants de Minecraft.

6. Prévisualisez et téléchargez

Utilisez le bouton d’aperçu pour vérifier blocs, objets et armures. Cliquez ensuite sur Télécharger le pack de ressources pour générer le ZIP localement dans votre navigateur.
