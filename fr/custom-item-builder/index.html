RUENFRDE

Créateur d'objets personnalisés Minecraft

Générez des commandes /give prêtes à l'emploi pour vos accessoires RP, récompenses de quêtes et objets liés au pack. Ce générateur adapte la syntaxe : restez compact sur les anciennes versions ou utilisez item_model, custom_data et les slots custom_model_data pour la 1.21.4+.

Ouvrir le générateur de packs Lire le guide CustomModelData

Configuration de l'objet

Utilisez l'objet vanilla que le joueur recevra (ex. paper, stick ou book).

Un ID logique pour les scripts, quêtes et vérifications admin. Il reste invisible pour les joueurs.

Qu'est-ce qui change selon la version ?

Utilisez la 1.21.4+ si vous souhaitez utiliser item_model, des états de texte lisibles ou des slots float, flag ou color dans custom_model_data.

Format ancien (jusqu'à 1.21.3)

Les anciens formats restent compacts : une valeur numérique CustomModelData, un nom visible, un lore et un ID caché optionnel.

1.21.4+ : item model et slots CMD

Les objets modernes séparent proprement le visuel de la logique : item_model pointe vers l'art, tandis que custom_model_data gère les slots float, string, flag et color.

Indiquez le chemin complet, par exemple cubeinsquare:item/path_here. Laissez vide si vous n'avez besoin que de l'ID caché sans modèle visuel.

Commande générée

Version : 1.21.4+. Longueur : 0 caractères.

Le générateur conserve volontairement la syntaxe plus ancienne compacte. En passant à la 1.21.4+, elle s'étend au format de composants plus riche.

Guild archive pass

1.21.4+ | minecraft:paper | 1 item

Cette version peut transporter simultanément une route de modèle visible, un ID caché lisible et des slots custom_model_data enrichis.

À quoi ça sert

  • Monnaies RP, permis, contrats, indices, clés, reliques et accessoires de guilde
  • Récompenses de quêtes nécessitant un ID logique caché et un modèle visuel distinct
  • Validation côté serveur où une identité d'objet lisible compte plus qu'un nombre aléatoire
  • Test rapide pour obtenir d'abord la commande, puis la pipeline de pack complète
