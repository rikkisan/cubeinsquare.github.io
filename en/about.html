About

Universal MC Pack is a small browser project for creators who want to build Minecraft resource packs without installing extra software or dragging in heavy frameworks.

Project Principles

Lightweight code

HTML, CSS, and JavaScript stay as simple files that can run locally or on GitHub Pages.

Practical tools

Every page should help users build, check, or understand a resource pack without unnecessary dependencies.

Easy to extend

New pages and utilities are added as separate files, so the project remains easy to maintain.
