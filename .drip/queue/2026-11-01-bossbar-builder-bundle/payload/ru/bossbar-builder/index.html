ENRUFRDE

Генератор bossbar для Minecraft

Создавайте наборы команд bossbar для ролевых событий, рейдов, ритуалов, отсчётов и общих предупреждений сервера. Выберите ID, заголовок, цвет, стиль, игроков и прогресс, а затем скопируйте чистую последовательность для командных блоков или datapack-функций.

Быстрые пресеты

Настройка bossbar

Bossbar лучше всего работает как состояние: опасность, заряд ритуала, фаза боя, время эвакуации или общее давление на сервере. Этот инструмент создаёт команды настройки; живой таймер всё равно нужно обновлять через повторяющийся командный блок или datapack-функцию.

Готовые команды

Bossbar

  • /bossbar add
  • /bossbar set value
  • /bossbar set players
  • /bossbar remove
