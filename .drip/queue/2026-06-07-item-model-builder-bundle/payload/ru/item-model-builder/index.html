ENRUFRDE

Minecraft Item Model Builder

Собирайте новые item definition JSON-файлы в assets//items/. Выберите тип ветвления, настройте fallback, добавьте кейсы и заберите готовый JSON без ручной сборки хрупкого каркаса.

Ветки

Готовый item definition

Где это помогает

  • Больше не нужно вручную собирать model/select/range_dispatch обёртку каждый раз.
  • Папка items остаётся читаемой для всей команды, а не живёт в случайных заметках.
  • Можно быстро собрать одну definition перед более крупной миграцией пака.

Перед экспортом

  • Проверьте, что пути к моделям уже существуют в ресурспаке.
  • Держите fallback безопасным, чтобы при пропущенной ветке не увидеть фиолетово-чёрную ошибку.
  • Если ветка завязана на custom_model_data, сверьте нужный индекс и слот со стороной команды предмета.
