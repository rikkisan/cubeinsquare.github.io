RUENFRDE

Minecraft Skin Editor

Classic skin painting in the browser: load a PNG, edit the 64x64 atlas, or paint directly on the 3D player model and watch the texture update in real time.

Read guide Back to tools

Painting tools

100%

Brush, eraser, and picker work both on the 2D atlas and directly on the model. Switch to Orbit when you only want to rotate the player.

Files

The editor accepts classic 64x64 skins and legacy 64x32 skins. Legacy files are expanded into the modern layout automatically.

Quick status

Atlas: 64 x 64 Tool: brush Brush: 1px

3D player preview

Paint right on the model when you want to check sleeves, the top of the head, boots, or the outer layer without guessing the atlas coordinates.

Classic 64x64 atlas

The atlas always stays in the vanilla Minecraft skin layout, so the exported PNG is ready for launcher use or any skin workflow that expects the standard template.

Fast workflow

1. Import or start fresh

Load an existing skin PNG or begin from the built-in starter skin to block out colors quickly.

2. Paint atlas or model

Use the atlas for precise pixel work and the model for sleeves, boots, hats, and any face you want to see in place.

3. Export the PNG

Download a standard Minecraft skin file and drop it into your launcher, website profile, or next skin iteration.
